# ShipBattle
Show off your tactics by managing your ship's abilities in Ship Battle!

## Inspirations

Before building Ship Battle, we played a few existing Battleship games online to get ideas
for what works well and what we want to improve on. Here are the two main ones we used:

---

### 1. [Battleship Online](https://www.battleshiponline.org/)

A simple browser-based Battleship game. It gave us a good baseline for how the core
game should feel and how the board layout should look. We used it to understand the
basic flow of placing ships and taking turns.

**What we liked:**
- Clean and easy to understand board
- Straightforward turn-based gameplay

**What we want to improve on:**
- No power-ups or abilities, gets repetitive fast
- No online multiplayer with real players

---

### 2. [Papergames Battleship](https://papergames.io/en/battleship)

A more polished online Battleship game that supports playing against random players,
friends, or a bot. It also has a gift/weapon system where power-ups can spawn on the
board mid-game.

**What we liked:**
- Online multiplayer with matchmaking
- Weapon system (big missile, nuclear missile, etc.) adds variety
- Clean UI with clear hit/miss indicators

**What we want to improve on:**
- Power-ups feel random and not tied to strategy
- No unique ship abilities, every ship plays the same
- UI could be more polished and modern
